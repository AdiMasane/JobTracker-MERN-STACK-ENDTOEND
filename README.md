MERN Job Tracker SaaS | Full Stack Project

Built a full-stack job application tracking platform using MongoDB, Express.js, React.js, and Node.js with JWT-based authentication.

Implemented features to create, update, delete, and manage job applications with statuses like Applied, Interview, Offer, and Rejected.

Developed a responsive dashboard with analytics, including application statistics and charts for tracking job progress.

Designed a secure user profile system allowing users to update personal information and passwords.

Integrated RESTful APIs for job and user management and deployed the application using Render with MongoDB Atlas.

Tech Stack: MERN, JWT, TailwindCSS, REST API, MongoDB Atlas, Render
